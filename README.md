# WhatsFlow Frontend

React + Vite CRM frontend for WhatsApp/SMS.

## Environment

| Environment | URL | API |
|-------------|-----|-----|
| Local | http://localhost:5173 | http://localhost:8000/api/v1 |
| Production | https://www.driveronhire.ai | https://api.driveronhire.ai/api/v1 |

## Deploy

Push to `main` triggers Vercel auto-deploy. Build: `npm run build`
